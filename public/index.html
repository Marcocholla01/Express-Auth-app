Auth IT

Welcome to the Auth IT! This project provides a robust authentication system for your applications.

Functions and Features:

  • User Registration: Allows users to create accounts with a username, email, and password.
  • User Login: Enables users to log in using their registered email and password.
  • Email Verification: Sends verification emails to confirm users' email addresses.
  • Password Reset: Provides functionality for users to reset their passwords if they forget them.
  • Session Management: Manages user sessions securely to keep users logged in.
  • JWT Authentication: Uses JSON Web Tokens for secure and scalable authentication.

Technologies Used:

  • HTML 5
  • CSS 3
  • Vanilla.js
  • MySQL
  • Express.js
  • Node.js
